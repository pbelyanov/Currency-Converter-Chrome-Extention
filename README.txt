Hello everyone. This is very simple currency converter that uses an API in order to fetch and provide real time currency conversion between 200+ currencies.

How to install:
1. Clone this repository and get it as a zip file.
2. Create a folder and extract the zip file in it.
3. Open Chrome browser and navigate to Options -> More tools -> Extentions
4. Enabled 'Develeoper mode' from the top right corner.
5. Select 'Load unpacked'and locate the folder that you just extracted from the zip.
